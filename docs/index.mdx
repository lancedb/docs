**LanceDB** is an open-source [multimodal lakehouse](https://lancedb.com/blog/multimodal-lakehouse/) for AI
designed to manage petabyte-scale data. It caters to a range of users:


<Steps>
  <Step title="Software developers and AI engineers">
    As a vector retrieval and memory layer for large-scale Generative AI and search applications leveraging
    LLMs and embeddings -- such as RAG, chatbots, agents, e-commerce search & recommendation systems, and more.
  </Step>
  <Step title="AI researchers, ML engineers and data scientists">
    As a data management platform for large-scale AI and machine learning workflows -- including
    model fine-tuning and training, feature engineering and explorative data analytics over
    petabyte-size datasets.
  </Step>
</Steps>

## Products

LanceDB offers a range of products to suit different needs, listed below.

With LanceDB Cloud , you can build secure, scalable, and high-performance vector search solutions. Manage your
Projects through the LanceDB Cloud UI , and connect via our simple REST API and SDKs in Python, Typescript and
Rust .

### LanceDB Enterprise

LanceDB Enterprise is both a **private cloud** or a **BYOC** solution that transforms your data lake into a
high-performance vector database or lakehouse that can operate at extreme scale.

<Note>
LanceDB Enterprise is designed for very large-scale workloads in private deployments, with
high performance at extreme scale,
or if you have strict security requirements, [contact us](mailto:contact@lancedb.com) to learn
more.
</Note>

### LanceDB OSS

LanceDB is also available as an open source embedded database. This is the
simplest way to get started with LanceDB, with client SDKs available in Python, TypeScript
and Rust. You can use LanceDB OSS to build search applications, RAG apps and more by running it
entirely locally on your filesystem.
